# __WeatherApp - A Java Swing Weather Application__

## __Description__

WeatherApp is a desktop application developed in Java, utilizing Swing for the GUI. It allows users to enter a city name and retrieve current weather information for that location, including temperature, weather condition, humidity, wind speed, and local time. The application fetches weather data from external APIs and displays it in an easy-to-understand format.

### __Features:__
+ Search by City: Users can search for weather information by entering a city name.
+ Current Weather Data: Displays temperature, weather condition (Clear, Cloudy, Rain, Snow), humidity, and wind speed.
+ Local Time: Shows the current local time of the searched city.
+ GUI: A user-friendly graphical interface that displays weather icons and information clearly.

### __Acknowledgements:__
